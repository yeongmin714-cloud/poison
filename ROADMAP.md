## Phase 3: 🌍 세계관 & 월드맵 디자인 (👑 NEW — 4방위 + 황제국)

> 게임 세계의 전체 구조를 설계합니다.
> **4개의 국가(동/서/남/북) × 각 20영지 + 중앙 황제국 = 총 81개 영지**
> **최종 목표: 황제국 탈환**

### 3.1 — 5개 지역 구성 & 난이도 방사형 구조

> **핵심:** 각 방위의 가장자리(외곽)에서 중앙(황제국)으로 갈수록 난이도가 점점 어려워집니다.

```
             [ 북 ]
     ┌─────────────────┐
     │ ⭐  ←   ⭐⭐⭐⭐ │  ← 바깥(쉬움) → 안쪽(어려움)
 [서] │  ┌───────────┐  │ [동]
     │  │  👑 황제국 │  │
     │  │ (최종목표) │  │
     │  └───────────┘  │
     │ ⭐  →   ⭐⭐⭐⭐ │
     └─────────────────┘
             [ 남 ]
```

#### 방사형 난이도 — 4개 링 구조

각 국가의 20영지는 황제국을 중심으로 **4개의 동심원 링(Ring)** 으로 배치:

| 링 | 위치 | 영지 수 | 설명 |
|:--:|:----|:------:|:------|
| 🟢 **Ring 1 (최외곽)** | 국가 가장자리 | 5개 | 병사 적음, 영주 입맛 단순, 초급 병사 |
| 🟡 **Ring 2** | 중간 바깥 | 5개 | 병사 보통, 영주 기본 입맛 |
| 🟠 **Ring 3** | 중간 안쪽 | 5개 | 병사 많음, 영주 까다로운 입맛 |
| 🔴 **Ring 4 (최내곽)** | 황제국 인접 | 5개 | 병사 최다, 영주 매우 까다로움 |

#### 링별 상세 난이도

| 항목 | Ring 1 (최외곽) | Ring 2 | Ring 3 | Ring 4 (황제국 인접) |
|:-----|:--------------:|:------:|:------:|:-------------------:|
| ⭐ **난이도 티어** | ⭐~⭐⭐ | ⭐⭐~⭐⭐⭐ | ⭐⭐⭐~⭐⭐⭐⭐ | ⭐⭐⭐⭐~⭐⭐⭐⭐⭐ |
| 👥 **병사 수** | 3~5명 | 6~10명 | 11~20명 | 21~40명 |
| 📈 **병사 레벨** | Lv.1~10 | Lv.11~20 | Lv.21~30 | Lv.31~40 |
| 🍽️ **영주 입맛** | 아무 음식 OK | 기본 요리 선호 | 고급 요리만 | 왕실급 최고 요리만 |
| 🧪 **영주 지병** | 없거나 가벼움 | 1개 지병 | 1~2개 지병 | 2~3개 복합 지병 |
| 🛡️ **방어력** | 낮음 | 보통 | 높음 | 매우 높음 |
| 💰 **점령 보상** | 작음 | 보통 | 큼 | 매우 큼 |

#### 지역별 링 난이도 가중치

4개 국가는 시작 난이도가 다르므로, 같은 Ring 번호라도 국가에 따라 실제 병사 수/레벨이 다릅니다:

| 국가 | Ring 1 병사 | Ring 2 병사 | Ring 3 병사 | Ring 4 병사 | 특징 |
|:----|:----------:|:----------:|:----------:|:----------:|:------|
| 🏁 **동 (East)** | Lv.1~3 (3명) | Lv.4~8 (5명) | Lv.9~14 (8명) | Lv.15~20 (12명) | 🟢 **초급, 시작 지역** |
| 🏁 **서 (West)** | Lv.3~6 (4명) | Lv.7~12 (7명) | Lv.13~18 (12명) | Lv.19~25 (18명) | 🟡 중상급 |
| 🏁 **남 (South)** | Lv.5~9 (4명) | Lv.10~15 (8명) | Lv.16~22 (15명) | Lv.23~30 (25명) | 🟠 고급 |
| 🏁 **북 (North)** | Lv.8~12 (5명) | Lv.13~20 (10명) | Lv.21~28 (18명) | Lv.29~40 (35명) | 🔴 **최고 난이도** |
| 👑 **황제국** | — | — | — | Lv.41~50 (50명) | **최종 목표** |

#### 5.3.10.4 — 병사 단독 공격 명령

> 플레이어가 직접 공격하지 않아도 동행 병사만 따로 지정하여 공격을 명령할 수 있습니다.

| 명령 방법 | 효과 |
|:----------|:------|
| 🟢 **드래그 선택 + 적 우클릭** | 선택 병사들이 대상에게 이동 후 공격 (플레이어는 공격 안 함) |
| 🔵 **병사 개인 우클릭** (미선택 상태) | 가장 가까운 병사 1명이 대상에게 이동 후 공격 |
| ⭐ **Ctrl+우클릭** | 모든 동행 병사가 대상에게 일제 공격 |

**명령 상태 표시:**

```
[동행 병사: 3명]
  🗡️ 전투 중 → 전리품 곰 Lv.5
  🚶 대기 중
  🗡️ 전투 중 → 적 병사 Lv.3
```

### 2️⃣ TDD — 반드시 컴파일 + 테스트 검증
- 코드 수정 후 **무조건** Unity batchmode로 컴파일 검증할 것
- `./run_tests.sh` 또는 직접 Unity -batchmode -executeMethod 실행
- 컴파일 오류 발생 시 → RED 상태. 수정 후 GREEN 될 때까지 반복
- **"Scripts have compiler errors" 메시지 없을 때까지** 수정을 멈추지 말 것
- 테스트 통과까지 완료해야 해당 작업이 ✅ 완료

### 3️⃣ 오류 수정 프로세스
```
오류 발견
  ↓
서브에이전트(delegate_task)에 오류 목록 전달
  ↓
서브에이전트가 파일별 수정 실행
  ↓
Unity batchmode 컴파일 재확인 (직접 실행)
  ↓
새 오류 없음 → 완료
  ↓
새 오류 있음 → 다시 서브에이전트에 전달 (반복)
```
