     │  │  👑 황제국 │  │
     │  │ (최종목표) │  │

### 38.1 — 영지별 특산물

| 🗡️ **무술 대회** | 군사 영지 | 용병 고용비 30% 할인, 경험치 2배 | 3분 |
| 🍺 **맥주 축제** | 선술집 영지 | 모든 음식 효과 1.5배, NPC 호감도 ↑ | 3분 |
| 🙏 **성지 순례** | 교회 영지 | 기부 효과 2배, 축복 지속시간 2배 | 3분 |
| 🎭 **가면 무도회** | 귀족 영지 | 변장 효과 ↑, 영주 접근 용이 | 3분 |
| 🔥 **불 축제** | 광산 영지 | 채광량 2배, 용광로 효율 ↑ | 3분 |

## Phase 39: 📋 퀘스트 체인 시스템 (✅ 완료)

> 단순 목록 → 선택지+결과+연쇄 퀘스트로 발전.

### 39.1 — 퀘스트 체인 구조

```
퀘스트 Chain = [퀘스트 1 → 선택 A → 퀘스트 2A]  or  [퀘스트 1 → 선택 B → 퀘스트 2B]
                          ↓                                    ↓
                   보상/결과 A                            보상/결과 B
```

### 39.2 — 퀘스트 종류

| 종류 | 설명 | 예시 |
|:-----|:------|:-----|
| 📋 **메인 퀘스트** | 스토리 진행 필수 | "왕의 복수 — 첫 번째 영주 처단" |
| 🗡️ **암살 의뢰** | 특정 NPC 제거 | "동쪽 3번 영주를 암살하라" |
| 🧪 **수집 퀘스트** | 특정 아이템 수집 | "늑대 이빨 10개를 모아와라" |
| 🏰 **영지 퀘스트** | 특정 영지 점령/방어 | "서쪽 7번 영지를 점령하라" |
| 🤝 **호의 퀘스트** | NPC 개인 부탁 | "영주 딸에게 선물을 전달하라" |
| 🕵️ **정보 수집** | 특정 정보 획득 | "북쪽 국경의 군사 정보를 입수하라" |
| 🎯 **도전 퀘스트** | 제한 조건 클리어 | "독만 사용하여 영주 3명 암살" |

### 39.3 — 선택지 시스템

```
[퀘스트: 길 잃은 영주 처리]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  "왕자님, 저 영주를 어떻게 하시겠습니까?"
  
  ⚔️ 1. 직접 처형한다 (무력, 호감도 변화 없음)
  ☠️ 2. 음식에 독을 넣는다 (은신, 발각 위험)
  🤝 3. 선처하고 내 편으로 만든다 (호감도 필요 50+)
  💰 4. 몸값을 받고 놓아준다 (금화 500 획득, 호감도 -20)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

| 선택 | 결과 | 이후 퀘스트 |
|:-----|:------|:-----------|
| 1. 처형 | 영주 사망, 병사 적대, 호감도 변동 없음 | "해당 영지 점령" |
| 2. 독살 | 영주 사망, 발각 시 전면전 | "해당 영지 점령 (쉬움)" |
| 3. 선처 | 영주 포섭, 아군 영지 획득 | "이웃 영지 공동 방어" |
| 4. 몸값 | 금화 획득, 영주 적대 유지 | "해당 영주 복수 퀘스트" |

### 39.4 — 퀘스트 연동 시스템
