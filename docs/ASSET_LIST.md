# 🎨 포이즌 (Poison) — 3D 에셋 목록

> **사장님이 Blender로 만들어서 GLB 파일로 제공해주실 에셋 목록입니다.**
>
> 📥 **파일 투입 폴더:** `Assets/Resources/Models/UserProvided/`
> - 여기에 `.glb` 파일을 넣어주시면 제가 Unity에 자동 연결합니다.
> - **파일 형식:** GLB (Binary glTF 2.0) — **휴머노이드 본(Bone) 포함 필수**
> - **스케일:** 1 Unit = 1 Meter, **방향:** Z축 = 정면
>
> 🤖 **애니메이션 처리 방식 변경 (2026-06-13):**
> - **Idle/Walk/Run/Jump/채집/제작/던지기/깃발꽂기/무릎꿇기** → Hermes가 Animation Rigging으로 자동 생성
> - 사장님은 **3D 모델(휴머노이드 본 포함)만 제공**하시면 됩니다
> - 아래 표에서 `🎬 Rig` = 코드 기반 애니메이션, `🎬 .anim 필요` = 직접 애니메이션 파일 필요
>
> 📖 **게임 데이터 참고:** `docs/GAME_DATA.md`
>

## 필수 본(Bone) 구조

모든 휴머노이드 모델에 아래 본이 포함되어야 Animation Rigging이 작동합니다:

```
Hips (루트)
├── Spine → Chest → Neck → Head
├── LeftUpperLeg → LeftLowerLeg → LeftFoot
├── RightUpperLeg → RightLowerLeg → RightFoot
├── LeftUpperArm → LeftLowerArm → LeftHand
└── RightUpperArm → RightLowerArm → RightHand
```
