# 🗣️ 포이즌 (Poison) — NPC 대사집

> **이 파일을 수정하면 게임 속 NPC 대사가 바뀝니다.**
> 각 NPC별로 대사 배열을 `- "대사내용"` 형식으로 추가/수정/삭제하면 됩니다.
> 새 NPC를 추가하려면 `## Phase X — [NPC 이름]` 섹션을 새로 만드세요.

---

## Phase 2 — 길 잃은 영주 (TutorialQuestNPC)

**NPC:** 추방지에서 만나는 첫 번째 영주. 튜토리얼 퀘스트 제공자.
**호출 키:** E (근접 시)
**퀘스트:** 배고프다 → 음식 만들기 → 설사약 → 10초 행동불능 → 영지 증서 획득

### 상태: 첫 대화 (NotStarted)

NPC를 처음 만나서 E 키를 눌렀을 때 — 순서대로 출력됨.

```yaml
- "어이, 젊은이! 나는 이 지역의 영주다."
- "길을 잃었는데... 배가 몹시 고프구나."
- "무언가 먹을 것을 구해다 주겠나?"
- "이 근처에 토끼와 멧돼지가 있으니 사냥을 해보게."
- "고기로 요리를 만들어 오게. 난 크래프트 테이블에서 기다리겠다."
```
→ 대화 종료 후 퀘스트 업데이트: **"음식을 만들어 영주에게 가져가라"**

---

### 상태: 음식 요구 중 (AskingFood)

플레이어가 음식을 가지고 와서 E 키를 눌렀을 때.

```yaml
- "오! 그 냄새! 정말 맛있어 보이는구나!"
- "...잠깐, 이 음식에 뭔가 들어있지 않나?"
- "(당신은 설사약을 넣은 것을 떠올린다)"
- "뭐... 아무것도 아니야. 자, 어서 먹게나."
```
→ 대화 종료 후 독 효과 발동

---

### 상태: 독에 걸림 (Poisoned)

음식을 먹고 10초간 행동 불능. 대화 불가 (자동 연출).

```yaml
- "윽... 배가...!!!"
- "이... 이 음식에 무슨 짓을 한 거냐!"
- "(10초 동안 행동 불능)"
```

---

### 상태: 사망 / 완료 (Dead / Complete)

10초 후 독 효과 종료. E 키로 영지 증서 획득.

```yaml
- "(영주가 쓰러져 있다)"
- "영지 증서를 가져가자..."
```

## 📝 편집 가이드

```yaml
# 새 NPC 추가 예시:
## Phase X — NPC 이름
- "첫 번째 대사"
- "두 번째 대사"
- "세 번째 대사"
```

1. 각 NPC는 **대화 상태(State)** 별로 나눠짐
2. 각 대사는 **순서대로** 출력됨
3. 플레이어가 E 키를 누를 때마다 다음 대사로 넘어감
4. 마지막 대사에서 E 키 누르면 대화 종료 + 상태 전환
5. `(괄호 안 텍스트)`는 **나레이션/설명** (화면에 표시됨, NPC 이름 없음)
6. 대사 중간에 `숫자초` 같은 변수는 스크립트에서 자동 치환
