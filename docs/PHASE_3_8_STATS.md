## 📌 개요

플레이어의 행동(사냥, 채집, 제작, 퀘스트 등)에 따라 **경험치(EXP)** 가 쌓이고 **레벨업** 시 능력치가 성장하는 시스템.

---

## 📈 경험치 & 레벨업

### 1.1 — 경험치 획득처

| 활동 | 획득 EXP | 비고 |
|:-----|:--------:|:-----|
| 🐇 초반 몬스터 사냥 | 10~30 | 몬스터 티어에 따라 차등 |
| 🐗 중반 몬스터 사냥 | 50~100 | |
| 🐉 후반 몬스터 사냥 | 150~300 | |
| 🌿 약초 채집 | 2~5 | 등급에 따라 |
| 🧪 약물 제작 (성공) | 20~50 | 난이도에 따라 |
| 🍲 요리 제작 (성공) | 15~40 | 난이도에 따라 |
| ❌ 제작 실패 | 0 | 경험치 없음 |
| 💬 NPC 퀘스트 완료 | 50~200 | 퀘스트 난이도에 따라 |
| 🏰 영지 점령 | 500~2000 | 영지 난이도에 따라 |

### 1.2 — 레벨업 필요 경험치

```
Lv 1→2:   100 EXP
Lv 2→3:   250 EXP
Lv 3→4:   500 EXP
Lv 4→5:   800 EXP
Lv 5→6:  1,200 EXP
Lv 6→7:  1,700 EXP
Lv 7→8:  2,300 EXP
Lv 8→9:  3,000 EXP
Lv 9→10: 4,000 EXP
...
(이후 레벨당 +1,000씩 증가)
```

- 최대 레벨: **Lv.50** (추후 확장 가능)
- 레벨업 시 모든 스탯이 소폭 상승

## ⚙️ 스테이터스 종류

| 스탯 | 설명 | 레벨당 증가 |
|:-----|:------|:----------:|
| ❤️ **체력 (HP)** | 플레이어 최대 체력 | +5 (Lv1=100 → Lv50=345) |
| 🧪 **약물 제조 (Alchemy)** | 약효 지속시간 증가, 고급 레시피 해금 | +1% 성공률, +2% 지속시간 |
| 🍲 **요리 제조 (Cooking)** | 요리 효과 상승, 고급 레시피 해금 | +1% 성공률, +2% 효과 |
| 💬 **화술 (Speech)** | NPC 친밀도 보너스, 정보 획득 확률, 가격 조정 | +1 친밀도 보정 |
| ⚔️ **전투 (Combat)** | 몬스터 공격력 보정 (향후) | +1% 데미지 |

|:--------:|:-----|:----------:|:-----------:|
| Lv.1~10 | 기본 약물만 제조 가능 | +0~5% | +0~10% |
| Lv.11~20 | 중급 약물 해금 | +5~10% | +10~20% |
| Lv.21~30 | 고급 약물 해금 | +10~15% | +20~30% |
| Lv.31~40 | 희귀 약물 해금 | +15~20% | +30~40% |
| Lv.41~50 | 전설 약물 해금 | +20~25% | +40~50% |

| 레벨 구간 | 효과 | 성공률 보정 | 효과 보정 |
|:--------:|:-----|:----------:|:---------:|
| Lv.1~10 | 기본 요리만 가능 | +0~5% | +0~10% |
| Lv.11~20 | 중급 요리 해금 | +5~10% | +10~20% |
| Lv.21~30 | 고급 요리 해금 | +10~15% | +20~30% |
| Lv.31~40 | 희귀 요리 해금 | +15~20% | +30~40% |
| Lv.41~50 | 전설 요리 해금 | +20~25% | +40~50% |

### 2.4 — 화술 (Speech)

NPC와의 상호작용에 영향을 줍니다.

| 레벨 구간 | 효과 |
|:--------:|:------|
| Lv.1~10 | 기본 대화, 상점 기본 가격 |
| Lv.11~20 | NPC 친밀도 상승 보정 +1, 정보 획득 확률 +5% |
| Lv.21~30 | 상점 가격 5~10% 할인, 퀘스트 보상 10% 증가 |
| Lv.31~40 | 영주 대면 시 우호도 보정, 정보 획득 확률 +15% |
| Lv.41~50 | 영주 선물 성공률 +10%, 모든 NPC 친밀도 상승 보정 +3 |

## 🎲 제작 성공/실패 시스템

### 3.1 — 기본 성공률

| 재료 등급 | 기본 성공률 | 예시 |
|:---------|:----------:|:-----|
| ⭐ 기본 재료 | 90% | 기본 약초 + 기본 고기 |
| ⭐⭐ 일반 재료 | 75% | 중급 약초 + 일반 고기 |
| ⭐⭐⭐ 고급 재료 | 55% | 희귀 약초 + 고급 몬스터 재료 |
| ⭐⭐⭐⭐ 희귀 재료 | 35% | 영주급 재료 + 전설 몬스터 |
| ⭐⭐⭐⭐⭐ 전설 재료 | 15% | 최종 보스 재료 + 고대 유물 |

### 3.2 — 최종 성공률 계산

```
최종 성공률 = 기본 성공률 + 레벨 보정(Alchemy/Cooking)
              - 난이도 페널티 (어려운 레시피일수록)
```

- **레벨 보정:** `(해당 스탯 레벨) × 1%`
- **난이도 페널티:** 쉬움 0%, 보통 -5%, 어려움 -15%, 매우 어려움 -30%

### 3.3 — 실패 시 결과

| 실패 유형 | 확률 | 결과 |
|:---------|:---:|:-----|
| ❌ **실패 (재료 보존)** | 실패의 30% | 재료 그대로, 제작만 안 됨 |
| 💥 **실패 (재료 소멸)** | 실패의 50% | 재료 1개 소멸, 나머지 반환 |
| 🔥 **대실패 (전소)** | 실패의 20% | 모든 재료 소멸 |

### 3.4 — UI 표시

제작 UI에 다음과 같이 표시:

```
[제작] 강력한 독약 (필요레벨: Alchemy Lv.5)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
재료: 쓴풀 ⭐ + 가시덤불 ⭐⭐
성공 확률: 65%  (기본75% + 레벨보정5% - 난이도15%)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[제작하기]           [취소]
```
